# Flawless Implementation Corp. Website

A GitHub Pages-ready static website for **Flawless Implementation Corp.**, a technology consulting firm focused on:

- Database systems architecture
- Database security
- Performance optimization
- Infrastructure stability
- Ongoing maintenance and improvement

## Files Included

- `index.html` — Main website page
- `assets/style.css` — Styling
- `assets/script.js` — Mobile navigation behavior

## How to Use

### Option 1: Upload to a GitHub repository
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/root`
5. Save the settings.
6. GitHub will publish the site and give you a live URL.

### Option 2: Test locally
Open `index.html` in your browser.
